# TreeHub Frontend

>This is the React/Vite frontend for TreeHub, a private marketplace and lending platform for Trees Residences.

## Features
- Modern React UI (Vite)
- Emoji chat, image upload, notifications
- Connects to backend API

## Local Development

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. (Optional) Set `VITE_API_URL` in `.env` to your backend URL for production. Leave empty for local dev.
3. Start the dev server:
   ```bash
   npm run dev
   ```

## Deployment
- Deploy on Vercel, Netlify, or similar static hosting.
- Set `VITE_API_URL` in `.env` to your backend’s public URL.

## Project Structure
- `components/` — UI components
- `pages/` — App pages
- `lib/` — Utilities
